## npm
```
包管理
npm install <Module Name>

全局安装与本地安装
npm install express          # 本地安装，当前目录下
npm install express -g   # 全局安装

查看某个模块的版本号
npm list grunt

查看所有全局安装的模块
npm list -g

```
