### 特殊符号

#### =>

```
https://nanxiao.me/scala-function-instance-sugar/
Scala中的=>符号可以看做是创建函数实例的语法糖。例如：A => T，A,B => T表示一个函数的输入参数类型是“A”，“A,B”，返回值类型是T。
另外，() => T表示函数输入参数为空，而A => Unit则表示函数没有返回值。
```
